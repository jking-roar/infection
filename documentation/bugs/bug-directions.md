Bug Directions
---

Bugs found will be placed here to be addressed at a later time.

Folder structure:


    /documentation/bugs
    .__ bug-directions.md
    .__ 20260207-example-bug-report.md


When making a bug report, please follow the format of the example bug report. The file name should be in the format of `YYYYMMDD-bug-title.md` where `YYYYMMDD` is the date the bug was found and `bug-title` is a short description of the bug.

No fixes should be made; only the bug report should be created. And committed.
